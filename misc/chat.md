# Chat

Make the main chat italics
`/me test`

Output: <i>* Peasant Tempest: dances</i>

## Direct Message

`/message <player>:` or `/m <player>:`

Example:
`/m Tempest: hello!`

## View a player profile

`/view <player>`

## Adding Items to Chat
Shift-click on an item will link it in your chat:

For most items, it'll appear like this:
`[item:artisan_shard]`

For trinkets and jewels, which are unique per item, it'll look like this:
`[jewel:118792]`
